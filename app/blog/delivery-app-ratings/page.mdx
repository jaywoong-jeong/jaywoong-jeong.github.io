# 배달앱 별점에 관해서.

별점은 신호이면서 동시에 게임의 규칙입니다.

## 왜곡

- 4.8과 4.6의 간극은 체감보다 큽니다: 노출, 검색, 쿠폰 정책이 달라집니다.
- 악천후·피크타임 변동성은 별점에 과도하게 반영됩니다.

## 인센티브

- 사장님은 안전한 메뉴·포장에 과투자하고, 실험은 줄어듭니다.
- 고객은 할인·프로모션에 더 민감하게 반응합니다.

## 개선 아이디어

- 컨텍스트 분리: 시간대·날씨·거리별 가중치를 적용합니다.
- 다중 지표: 재주문율, 문제 해결 속도, 포장 안전 지표를 함께 노출합니다.
- 리뷰 프레이밍: 불만 접수 플로우와 공개 리뷰를 분리합니다.

별점은 편하지만, 좋은 경험의 전부는 아닙니다. 신호를 더 풍부하게 만들어야 합니다.
